# Lightly Colored Sea Lanterns (Forge)

Lightly Colored Sea Lanterns is a Minecraft mod that add 16 colored Sea Lanterns based on the dye available in the game.

### Craft  
S = Sea Lantern  
D = Dye

S S S  
S D S = Dyed Sea Lantern Block  
S S S
